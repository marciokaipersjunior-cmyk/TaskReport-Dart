# TaskReport Dart

## Sobre o projeto

O TaskReport Dart é um simulador de tratamento de dados de tarefas em Dart.

O projeto simula o retorno de uma API por meio de uma lista de mapas,
transforma esses dados em objetos e gera relatórios com base nos status das
tarefas.

## Objetivo

Praticar os principais conceitos do Módulo 01:

 lógica de programação;
 linguagem Dart;
 tipos de dados;
 estruturas de decisão;
 estruturas de repetição;
 funções;
 arrow functions;
 Lists;
 Maps;
 Sets;
 métodos de coleção;
 classes;
 objetos;
 atributos;
 métodos.

## Como executar

Execute o arquivo `main.dart` no ambiente utilizado em aula.
Exemplo:

```bash
dart run main.dart
```
